## CH13-1 -- github context object
`github` context object contains data object of current workflow environment and webhook event payloads.

In a workflow, you can know these data from accessing `github` context object.

  + webhook event payloads that current workflow recieved

     - which event triggers the workflow?
     - who triggers the event?

  + current branch of current workflow
  + owner of the repo of current branch
